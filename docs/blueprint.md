# **App Name**: Apex Wraps

## Core Features:

- Project Portfolio Display: Showcase past car wrapping and customization projects with high-quality images, descriptions, and categorization for easy browsing.
- Custom Quote Request Form: Allow potential clients to submit detailed inquiries and requests for custom wrapping services, capturing essential project specifications and contact information for lead generation.
- AI Design Suggestion Tool: Provide a generative AI tool where users can receive custom wrap design ideas or color scheme suggestions tailored to their vehicle type or aesthetic preferences.
- Comprehensive Service Catalog: Present detailed information about all available wrapping services, materials, pricing structures, and the customization process.
- Admin Content Management: A secure, administrative interface to allow 'Apex Wraps' staff to easily add, edit, and remove projects, services, and other website content via a Firebase backend.
- User Authentication: Implement user authentication using Firebase to secure administrative access to the content management system.

## Style Guidelines:

- Primary accent color: Bold Orange (#FF4D00). This vibrant hue will be used for calls-to-action, key navigational elements, and brand accents, reflecting the brand's energetic and performance-oriented aesthetic.
- Background color: Deep Black (#0A0A0A), as explicitly requested. This provides a high-contrast, modern canvas, making the vibrant orange and content stand out distinctly.
- Secondary accent color: Energetic Magenta-Red (#EE5E7A). Derived as an analogous hue to the primary orange (HSL: H=350, S=80%, L=60%), this color will be sparingly used to provide an additional layer of visual interest and distinction.
- Headline Font: 'Bebas Neue' (sans-serif, display) for impactful hero headlines and prominent titles, conveying a dynamic and professional brand presence. Note: currently only Google Fonts are supported.
- Subheading Font: 'Syne' (sans-serif, weights 700-800) for strong, precise subheadings that maintain visual hierarchy and complement the main headlines. Note: currently only Google Fonts are supported.
- Body Font: 'DM Sans' (sans-serif, weights 300-500) for clean, highly readable body text, ensuring an accessible and modern user experience. Note: currently only Google Fonts are supported.
- Utilize minimalist, geometric line icons that align with modern automotive design, precision tools, and digital interface aesthetics.
- Section Label Component: A reusable UI component that displays a small, uppercase orange label with an accompanying short orange line to its left, serving as a distinct visual cue before each content section.
- Button Component: A versatile button component offering two variants: a 'primary' style with an orange filled background and distinctive chamfered clip-path corners for key interactions, and an 'outline' style featuring a transparent background with a white border, transforming to orange on hover for secondary actions.
- Subtle and swift transition animations for state changes and user interactions, providing a polished and responsive feel without distracting from the content.
